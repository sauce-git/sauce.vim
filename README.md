# 🚀 Sauce's Neovim Setup

Welcome to my personal Neovim configuration!

This setup is crafted for a fast, modern, and delightful development experience on Linux.

---

## ✨ Features

- ⚡ **Modular Lua configuration** — Clean, organized, and easy to extend
- 📦 **Plugin management** — Powered by [lazy.nvim](https://github.com/folke/lazy.nvim) or [packer.nvim](https://github.com/wbthomason/packer.nvim)
- 🧠 **LSP, Treesitter, and Autocompletion** — Intelligent code navigation, highlighting, and completions out of the box
- 🎛️ **Custom keymaps & UI enhancements** — Streamlined workflow and beautiful visuals

---

## 🚀 Getting Started

1. **Clone this repo:**

   ```sh
   git clone https://github.com/yourusername/nvim-config.git ~/.config/nvim
   ```

2. **Launch Neovim:**

   ```sh
   nvim
   ```

3. Plugins will install automatically on first launch.
